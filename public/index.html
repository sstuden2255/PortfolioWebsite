Projects

VAST

A location-based multiplayer gaming application for iOS.

VAST is a fun and interactive way to play the game of infection, a modified game of tag with two teams; Infected and Humans. Utilizing a built-in map to view the real-time locations of other players, infectors fight to find and infect humans, while humans fight to stay alive! A group project for my Mobile Software Capstone.

View GitLab Repo

Thrive

A gamified productivity and social media web application.

Thrive is an application that promotes productivity through task prioritization and social encouragement. Key features include account creation, a task dashboard, profile customization, a calendar page, and a community page. A group project for my Software Engineering class.

View Github Repo View Figma Design

Motor Nation

A pseudo ecommerce website for purchasing Grand Theft Auto vehicles.

Motor Nation is a website where you can find and purchase GTA Vehicles of all shapes and sizes. Key features include searching, filtering, user account creation, a shopping cart, bulk purchases, order history, and customer reviews. A pair project for my Web Programming class.
